# Base React Project

Boilerplate project to start a simple prod-ready React Application

## Stack

- jest
- react
- redux
- babel
- parcel
- eslint

## Usage

### Start project

```sh
npm start
```

### Lint

#### Format code

```sh
npm run lint # lint and format code
```

#### Check code styles

```sh
npm run lint-check # check code lint
```

### Production build

```sh
npm run build
```
